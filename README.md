## 📌 Basic Example

If you place one tag inside another, the app helps you see whether that nesting makes sense.

Example:

## 🧩 HTML Nesting Tips

Here are a few simple rules that help when you work with HTML:

- Keep block content in the right container
- Avoid placing the same element inside itself unless the tag allows it
- Use lists with the correct list items
- Match opening and closing tags
- Keep text tags inside elements that support text

These habits make your page easier to read and easier to fix.
